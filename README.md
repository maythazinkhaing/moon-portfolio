# 🌙 Moon Portfolio

Personal portfolio for **May Thazin Khaing (Moon)** — web developer based in Bangkok, Thailand.

Built with Vite · React 18 · TypeScript · Tailwind CSS v3

---

## Features

- Night / Sunset theme toggle with smooth crossfade
- Floating glass dock navigation
- Browser-window-style modal for each section
- Singing chibi mascot with Web Audio animations
- Fully responsive (mobile, tablet, desktop)

---

## Running Locally

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # production build → dist/
```
